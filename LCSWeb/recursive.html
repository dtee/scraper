Recursive Algorithm

int lcs_length(char * A, char * B) {
       if (*A == '\0' || *B == '\0')
              return 0;
       else if (*A == *B)
              return 1 + lcs_length(A+1, B+1);
       else
              return max(lcs_length(A+1,B), lcs_length(A,B+1));

}

This solution is correct because it implements the Theorem but it’s really time consuming and makes a lot of recursive calls. If string A’s length (n) equals string B’s length (m), then there would be 2^n possible subsequences. Based on this, this recursive solution takes exponential time. Notice this solution contains optimal sub structure; its recursively comparing something is compared in the previous recursion call. We can use dynamic programming technique and optimize this.

LCS-Delta

LCS-Delta(X,Y) {
       m <- LENGTH[X];
       n <- LENGTH[Y];
       for i <- 1 to m, do
              c[i,0] <-0;
       for j <- 0 to n, do
              c[0,j] <-0; b <- c;
      
       for i <- 1 to m, do {
              for j <- 1 to n do {
                     if (xi = yj) {
                           c[i,j] <- c[i-1,j-1]+1;
                           b[i,j] <- "UP-LEFT";
                     }
                     else if (c[i-1,j] >= c[i,j-1]) {
                           c[i,j] <- c[i-1,j];
                           b[i,j] <- "UP";
                     }
                     else {
                            c[i,j] <- c[i,j-1];
                           b[i,j] <- "LEFT";
                     }
              }
       }
       return c and b

}

This solution is the most popular dynamic solution for finding the Longest Common subsequence. This solution creates two matrixes; one matrix stores the LCS information while the other matrix stores the LCS sequence finding information. Also, notice this solution starts from the beginning the string, at the position 0. Using the b matrix, you can trace back the longest common sequence. This solution is better than the last solution because easier to build the LCS sequence from the result matrix, and it’s easier to follow the arrows to find a sub common sequence beginning at any given character in string. However the complexity is still the same O(nm).

Improvements

In this algorithm, we can get rid of the b table. To find c[I,j] entry, we only need to know c[i-1, j-1], c[i-1, j] and c[i, j-1]. Give c[i,j], we can find those three values without needing the need to construct table b. With this, we can get rid of the b table saving us Θ(nm) space.

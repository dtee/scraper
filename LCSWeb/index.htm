Introduction

What is longest common subsequence?

Given a long string and short string, the letters from short string that’s contained in order (but possibly separated) within the long string would be called subsequenceThe common subsequence would be the letters in two given text that shares a common subsequence.

For example, given two string: “abcbdab” and “bdcaba”, it can have many sub sequences: “bcab”, “bdab”, “db”, “ab”, ”aba”, etc. In this case the longest common subsequence would be “bcba” or “bdab” with the length of 4, since there is no common subsequence of length 5 or greater.

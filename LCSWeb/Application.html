Applications

  • Data Files – Application that compare two files such as Unix’s “diff” implements LCS algorithm.
  • Screen Update – Applications that requires efficient screen update uses LCS to find the changes to screen so that only the changes are send from server. Such applications includes Unix’s “emacs”.
  • DNA – Biologist to use LCS to compare the similarity between two species’ DNA.
